MASE 麦哲思AI软件开发统一流程

Measures AI Software Engineering — v1.1 — 四Agent · 六阶段 · PDCA 闭环

快速开始

# 1. 安装框架
cd measures-framework-package && ./install.sh

# 2. 创建项目
mase init my-project -p my_app -c auth payment

# 3. 进入项目
cd my-project
pip install -e .

# 4. 对 AI 说: "创建新项目" — 六阶段自动运转

六阶段流程

1. Proposal P

Agent 2 — 需求

brainstorming → 原型 → proposal.md → 测试用例

2. Design P

Agent 3 + 4

技术预研 → 架构 → 详细设计 → specs → 设计评审

3. Build D

Agent 3 + 4

TDD 10步微循环 → 逐个 Scenario 增量构建

4. Verify C

Agent 4

端到端验证 → bug-fixer 微循环 → 全部 BUG 关闭

5. Retro A

Agent 4 + 1

复盘分析 → 模式识别 → 改进流程

6. Release

Agent 1

合规审查 → 使用手册 → commit → 归档

四Agent架构

Agent 1 计划与统管

接收需求 → 分解任务 → 调度 A2/A3/A4 → 门禁管理 → Release

Skills: project-planning-expert, git-commit

Agent 2 需求

需求澄清 → 原型确认 → 编写 proposal.md + 验收标准

Skills: brainstorming, frontend-skill

Agent 3 开发

技术预研 + POC → 架构设计 → specs → TDD 构建

Skills: test-driven-development, webapp-testing

Agent 4 质量

设计评审 → 代码评审 → 安全扫描 → Bug修复

Skills: code-quality-controller, code-review, security-review, bug-fixer
